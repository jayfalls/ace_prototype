# Dependencies

- Before running install
    - [Podman](https://podman.io/docs/installation)
    - [Ollama](https://ollama.com/download)

# Usage

```shell
./ace
```

## Arguments

- `-s` Stop the ACE
- `-r` Restart the ACE
- `-u` Update the ACE
- `--dev` Run the ACE in development mode
- `--test` Run the ACE tests
