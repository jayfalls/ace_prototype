# Design

## Visual Style

### Color Palette
- **Primary**: #6366f1 (Indigo)
- **Secondary**: #8b5cf6 (Violet)
- **Accent**: #10b981 (Emerald)
- **Background**: #0f172a (Slate 900)
- **Surface**: #1e293b (Slate 800)
- **Text Primary**: #f8fafc (Slate 50)
- **Text Secondary**: #94a3b8 (Slate 400)
- **Error**: #ef4444 (Red)
- **Success**: #22c55e (Green)

### Typography
- **Font Family**: system-ui, -apple-system, sans-serif
- **Headings**: Bold, 1.5rem - 2rem
- **Body**: Regular, 1rem
- **Code**: monospace, 0.875rem

### Layout
- **Sidebar**: Fixed left, 240px width
- **Content**: Fluid, max-width 1200px
- **Cards**: Rounded corners (8px), subtle shadow
- **Spacing**: 8px base unit

### Components
- **Buttons**: Primary (filled), Secondary (outline), Danger (red)
- **Inputs**: Dark background, light border, focus ring
- **Cards**: Elevated surface with subtle border
- **Modal**: Centered overlay with backdrop blur

### Animations
- **Transitions**: 200ms ease-out
- **Page transitions**: Fade
- **Loading**: Pulse animation on skeletons
