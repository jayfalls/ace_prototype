# Changelog Entry Template

Use this template for each day's changelog entry.

## Format
- File: `YYYY-MM-DD.md`
- Date in header should match filename

## Template
```markdown
# YYYY-MM-DD

## Changes

### Added
- [Description of additions]

### Changed
- [Description of changes]

### Fixed
- [Description of fixes]

### Removed
- [Description of removals]

## Notes
[Any additional notes about today's changes]
```
